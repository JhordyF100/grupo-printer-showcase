GRUPO PRINTER SHOWCASE - VERSIÓN BASE OFFLINE

Para activar la rotación 360° de cada equipo:
1. Entra a img/productos/
2. Crea una carpeta con el ID del equipo, por ejemplo: ma5500ifx
3. Guarda 24 fotos numeradas así:
   01.jpg, 02.jpg, 03.jpg ... 24.jpg
4. Abre index.html.

El catálogo funciona sin internet siempre que todos los archivos estén en la misma carpeta.
